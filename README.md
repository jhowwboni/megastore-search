# Sistema de Busca Otimizado - MegaStore

## 📌 Descrição
Sistema desenvolvido em Rust para otimizar buscas em um catálogo de produtos utilizando tabelas hash.

## 🚀 Funcionalidades
- Busca por nome
- Busca por categoria
- Indexação eficiente

## 🛠 Tecnologias
- Rust
- Cargo

## ▶️ Como executar

```bash
cargo run
